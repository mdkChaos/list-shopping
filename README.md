# List Shopping

A shopping list planning system that allows users to create, edit, and delete shopping lists and their items. The application also supports list sharing and maintains a history of changes.

## Project Structure

- **/api** — The backend of the project, implemented with Laravel 11 using MySQL and Laravel Passport.
- **/frontend** — The frontend of the project, implemented with Vue.js 3.

## `/api` Folder

This subproject contains the backend part of the application, including the API for interacting with the database, user authentication, and managing shopping lists.

## `/frontend` Folder

This subproject contains the frontend part of the application, including the user interface for interacting with the API.
